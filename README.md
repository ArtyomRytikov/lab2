**И-1. Построение гистограммы** (рейтинг: 10)

**И-3. Приоритезация задач на основе вычисления частотных характеристик инцидентов** (рейтинг: 5)
- Реализовано в файле `src/core/tasks/task_prioritizer.h`

**И-9. Поиск наиболее частых подпоследовательностей** (рейтинг: 12)
- Реализовано в файле `src/core/tasks/most_frequent_substrings.h`
